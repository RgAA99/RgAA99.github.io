补充学习

linux

  1. 常见目录

    • /usr 是unix shared resources共享资源，很多应用程序都放在这个目录下
    • /usr/bin 用户使用的应用程序
    • /usr/sbin 超管使用的应用程序
    • /usr/local 用户级程序目录，理解为C:/Progrem Files/，存一些用户手动安装的软件的默认目录
    • /opt 默认是空的，一般用于装一些额外的软件，理解为D://software
    • /var 存放经常修改的数据，如/var/log
    • / 为根目录，也是超管的主目录
    • ～ 为用户的家目录 在linux上为/home/roger，在mac上为/Users/roger

  2. 常见命令

    • cat
    • open/vim
    • export
    • source
