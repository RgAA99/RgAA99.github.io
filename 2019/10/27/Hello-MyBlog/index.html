#一、什么是hexo
基于nodejs的静态网页生成器，可以用来写博客，展示产品等。

###安装过程

  1. 安装node和git
    • git是为了从hexo库中拉取以及将本地部署推到github.io
  2. 按命令执行
    • 安装hexo和相关命令
      npm install -g hexo-cli
    • 在指定下初始化hexo
      hexo init <folder>
    • 报错后进文件夹重新install
      cd <folder>
      npm install
    • 生成网站页面(生成在public目录下)
      hexo g
    • 部署至本地server
      hexo s
  3. 写作
    • 创建一篇新文章(默认模板是post,文章的模板参照scaffolds下的md文件)
      hexo new "文章标题"
    • 生成一个草稿(不会被渲染和生成到网页上去)
      hexo new draft "草稿标题"
    • 然后生成一个网页
      hexo g
      hexo s
  4. 配置(分为网站_config.yml和主题_config.yml)

  1. 推到远端

###(2) git bash下ctrl+C无效，无法终止node

  • 方式一：git的bug，回退至2.10.0
  • 方式二：windows下kill一个进程
    1. 先查看端口被哪个进程所占用？
      netstat -aon|findstr "4000"
    2. 杀掉这个进程（必须要跳到cmd中才能执行成功）
      taskkill /pid 9352 -t -f
    3. 检查是不是被杀掉了
      tasklist|findstr "9352"
